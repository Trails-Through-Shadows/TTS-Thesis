\CzechAbstract{Tato práce se zabývá tvorbou aplikačního rozhraní pro hybridní výpravnou hru, které poskytuje koncové body pro administrativní rozhraní a uživatelské prostředí. Pro implementaci je použit \gls{framework} Spring Boot pro programovací jazyk Java a za pomocí rozhraní Swagger je možné výsledné \gls{api} testovat. Jako úložiště je vybrána databáze MariaDB založená na MySQL. Finální produkt představuje aplikační rozhraní, které bude poskytovat koncové body ostatním spoluřešitelům. Společně s ním vznikla i příručka popisující jednotlivé koncové body.}

\CzechKeywords{API; REST; RESTful; SOAP; GraphQL; JSON; HTTP; Databáze; MariaDB; výpravná hra; vývoj hry; Java; Spring Boot; }

\EnglishAbstract{This thesis deals with the creation of an application interface for a hybrid narrative game that provides endpoints for the administrative interface and user environment. The \gls{framework} Spring Boot for Java programming language is chosen for implementation and using the Swagger interface the resulting \gls{api} can be tested. The database MariaDB based on MySQL are chosen as the storage. The final product is an application interface that provides endpoints to other co-researchers. Together with this application interface, a manual describing the endpoints is produced.}

\EnglishKeywords{API; REST; RESTful; SOAP; GraphQL; JSON; HTTP; Database; MariaDB; adventure game; game development; Java; Spring Boot;}
